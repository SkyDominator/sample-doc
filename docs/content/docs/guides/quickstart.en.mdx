# Quickstart

This guide explains the shortest path to opening a mock device session and running a single generation request with the `nano-llm-engine` SDK.
Instead of real RNGD hardware, it simulates device integration through the `MOCK_RNGD_HARDWARE` environment variable.

## 1. Install the SDK

Install the Python SDK from the project root.

```bash
pip install -e ./sdk
```

## 2. Minimal Run Example

The example below shows the basic flow from engine creation through device load and a single generation request.

```python
import os

from nano_llm import InferenceConfig, KVCacheConfig, NanoLLMEngine

os.environ["MOCK_RNGD_HARDWARE"] = "true"

engine = NanoLLMEngine(
	model_path="models/mock-llm.bin",
	quantization="INT8",
	kv_cache_config=KVCacheConfig(max_seq_len=4096, cache_dtype="fp16"),
)

engine.load_to_device(device_id=0)

result = engine.generate(
	prompt="Explain how the INT8 setting and token budget appear in the output string.",
    config=InferenceConfig(batch_size=1, max_tokens=64, temperature=0.2),
)

print(result)
```

Call `engine.unload()` when you are ready to close the session.

## 3. Streaming Inference Example

The streaming interface returns the synchronous generation result split into whitespace-delimited tokens.

```python
import os

from nano_llm import InferenceConfig, NanoLLMEngine

os.environ["MOCK_RNGD_HARDWARE"] = "true"
engine = NanoLLMEngine(model_path="models/mock-llm.bin", quantization="FP16")
engine.load_to_device()

stream = engine.generate_streaming(
	"Summarize which settings the streaming output was generated from.",
    InferenceConfig(max_tokens=24, temperature=0.0),
)
for token in stream:
    print(token)

engine.unload()
```

## 4. Check a Failure Scenario

If the hardware environment variable is missing, the SDK raises an error immediately and exposes the missing runtime setup before a device session starts.

```python
import os

from nano_llm import NanoLLMEngine
from nano_llm.errors import DeviceNotFoundError

os.environ.pop("MOCK_RNGD_HARDWARE", None)

engine = NanoLLMEngine(model_path="models/mock-llm.bin")
try:
	engine.load_to_device(0)
except DeviceNotFoundError as exc:
	print(f"Expected error: {exc}")
```
