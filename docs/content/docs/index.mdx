# RayKimLLM

`RayKimLLM`는 모의 LLM 추론 경로를 작은 코드베이스로 정리한 예제 프로젝트입니다.
핵심 구성은 Python SDK의 엔진 수명주기, 요청 설정 객체, 장치 메모리 추상화, 그리고 이 동작을 확인하는 테스트입니다.

## 핵심 구성 요소

- `RayKimLLMEngine`: 장치 로드, 동기 생성, 스트리밍 생성, 메모리 조회, 언로드를 담당합니다.
- `InferenceConfig`: 배치 크기, 토큰 예산, temperature 같은 요청 단위 설정을 보관합니다.
- `KVCacheConfig`: KV cache 길이와 정밀도 라벨을 엔진 수준 설정으로 유지합니다.
- `DeviceMemory`: Rust 확장이 있으면 이를 사용하고, 없으면 Python 구현으로 동작하는 메모리 풀 래퍼입니다.
